Del 2: For-løkke med indeks

Når du skriver:
for (i in numbers.indices) {
    println(i)
}

så er i indeksen, altså plasseringen i listen.
Eksempel:

val numbers = listOf(10, 20, 30)
for (i in numbers.indices) {
    println(i)
}

Output:

0
1
2

Ikke 10, 20, 30.

For å hente verdien bruker vi:

numbers[i]

Del 3: withIndex

Dette er ofte enklere å lese:

for ((index, value) in list.withIndex()) {
    println("$index: $value")
}

Da får du både indeks og verdi samtidig.

Oppgave 7

Hva skrives ut?

fun main() {
    val courses = listOf("IN1000", "IN2000", "IN2010")
    for ((index, course) in courses.withIndex()) {
        println("$index -> $course")
    }
}

Svar:
0 -> IN1000
1 -> IN2000
2 -> IN2010

Del 4: Bygge opp en sum med løkke

Dette ligner Python veldig.

Python:

sum = 0
for x in numbers:
    sum += x

Kotlin:

var sum = 0
for (x in numbers) {
    sum += x
}

Oppgave 9

Hva skrives ut?

fun main() {
    val numbers = listOf(1, 2, 3, 4)
    var sum = 0
    for (number in numbers) {
        sum += number
    }
    println(sum)
}

Svar:
10

fun countEven(numbers: List<Int>): Int = numbers.filter { it % 2 == 0 }.count()

Den skal telle hvor mange partall listen har.

Eksempel:

for (number in numbers) {
    if (number % 2 == 0) {
        count++
    }

Del 6: Lage ny liste med løkke

Dette tilsvarer filter, men med vanlig løkke.

Oppgave 12

Skriv funksjonen:

fun onlyEven(numbers: List<Int>): List<Int> {
    val newList = mutableListOf<Int>()
    for (num in numbers) {
        if (num % 2 == 0) {
            newList.add(num)
        }
    }
}

Den skal returnere en ny liste med bare partall.

Eksempel:

onlyEven(listOf(1, 2, 3, 4, 5, 6)) // [2, 4, 6]

Hint:

val result = mutableListOf<Int>()
for (number in numbers) {
    if (...) {
        result.add(number)
    }
