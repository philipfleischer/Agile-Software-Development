Oppgave 1: filter endrer ikke originalen

Hva skrives ut?

fun main() {
    val numbers = listOf(1, 2, 3, 4)
    numbers.filter { it > 2 }
    println(numbers)
}

Svar:
listOf(3, 4)

Oppgave 2: Riktig bruk av find

Fiks koden:

val names = listOf("Ole", "Ada", "Philip")
val result = names.find(it == "Ada")
println(result)

Svar:
val result = names.find { it == "Ada" }
println(result)

⸻

Oppgave 3: maxBy

Gitt:

data class Student(val name: String, val points: Int)
val students = listOf(
    Student("Ole", 20),
    Student("Ada", 35),
    Student("Philip", 30)
)

Skriv én linje som finner studenten med flest poeng.

Forventet:

Student("Ada", 35)


Svar:
val result = students.maxBy { it.points }
println(result)


⸻

Oppgave 4: map etter filter

Bruk samme students.

Lag en liste med navnene til studentene som har minst 30 poeng.

Forventet:

listOf("Ada", "Philip")

Svar:
val result = students.filter { it.points >= 30 }.map { it.name }
println(result)

⸻
