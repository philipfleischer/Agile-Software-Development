Oppgave 1: Output med filter/map/fold

Hva skrives ut?

fun main() {
    val numbers = listOf(1, 2, 3, 4, 5, 6)
    val result = numbers
        .filter { it % 2 == 0 }
        .map { it * 3 }
        .fold(0) { acc, x -> acc + x }
    println(result)
}

Svar:
etter filter: 2,4,6
etter map: 6,12,18
etter fold som samler alt til én verdi med startverdi 0:
acc = 0
0 + 6 = 6
6 + 12 = 18
18 + 18 = 36
output: 36

fun isReady(boreks: List<BorekInOven>): List<Borek> {

Oppgave 7: isSet uten toSet

Du har allerede løst isSet med toSet(). Nå skal du løse den med løkker.

fun isSet(numbers: List<Int>): Boolean {
    val newSet = mutableSetOf<Int>()
    numbers.forEach { newSet.add(it) }
    return numbers.size == newSet.size
}

Den skal returnere false hvis samme tall finnes mer enn én gang.

Eksempel:

isSet(listOf(1, 2, 3))       // true
isSet(listOf(1, 2, 2, 3))    // false

Hint: Du kan bruke en mutableListOf<Int>() eller mutableSetOf<Int>() til å huske tall du har sett før.
